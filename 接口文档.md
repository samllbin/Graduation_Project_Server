# app_server 接口文档（全量）

> 基于当前 `src/**/*controller.ts` 代码整理。
> 说明：项目多数接口返回统一 JSON 结构，`code` 为业务码（不等同 HTTP 状态码）。

---

## 1. 通用说明

### 1.1 返回格式（大多数接口）
```json
{
  "code": 200,
  "message": "xxx",
  "data": {}
}
```

### 1.2 鉴权方式
需要鉴权的接口使用 `AuthGuard`，请求头：
```http
Authorization: Bearer <access_token>
```

当前鉴权除 JWT 外，还校验登录 IP 一致性（登录时写入 Redis，后续请求校验）。

## 4. Email 模块

### 4.1 发送验证码
- **POST** `/email/send-code`
- 鉴权：否
- Body:
```json
{
  "email": "user@example.com"
}
```

### 4.2 校验验证码
- **POST** `/email/verify-code`
- 鉴权：否
- Body:
```json
{
  "email": "user@example.com",
  "code": "123456"
}
```

## 5. Auth 模块

### 5.1 登录
- **POST** `/auth/login`
- 鉴权：否
- Body:
```json
{
  "userName": "test",
  "password": "123456"
}
```
- 成功返回 `access_token`、`refresh_token`、用户信息。

### 5.2 刷新短 token
- **POST** `/auth/refresh`
- 鉴权：否
- Body:
```json
{
  "refresh_token": "<refresh_token>"
}
```
- 成功返回新的 `access_token`。

## 6. User 模块

### 6.1 注册用户
- **POST** `/user/create`
- 鉴权：否
- Body:
```json
{
  "userName": "test",
  "password": "123456",
  "email": "user@example.com",
  "code": "123456",
  "avatar": "optional",
  "signature": "optional"
}
```

### 6.2 查询用户（按用户名）
- **GET** `/user/:userName`
- 鉴权：否
- Path:
  - `userName` 用户名

### 6.3 修改个人资料
- **PATCH** `/user/profile`
- 鉴权：是
- Body（至少一个字段）:
```json
{
  "avatar": "optional",
  "signature": "optional"
}
```

### 6.4 登录态修改密码
- **PATCH** `/user/password`
- 鉴权：是
- Body:
```json
{
  "currentPassword": "old",
  "newPassword": "new123456"
}
```

### 6.5 忘记密码-发送验证码
- **POST** `/user/password/forgot/send-code`
- 鉴权：否
- Body:
```json
{
  "email": "user@example.com"
}
```

### 6.6 忘记密码-重置密码
- **PATCH** `/user/password/forgot/reset`
- 鉴权：否
- Body:
```json
{
  "email": "user@example.com",
  "code": "123456",
  "newPassword": "new123456"
}
```

## 7. Upload 模块

### 7.1 上传图片
- **POST** `/upload/image`
- 鉴权：是
- Content-Type: `multipart/form-data`
- FormData:
  - `file`（图片文件）
- 成功返回：
  - `fileName`
  - `filePath`
  - `originalName`
  - `url`

## 8. Post 模块

### 8.1 发帖
- **POST** `/post/create`
- 鉴权：是
- Body:
```json
{
  "title": "optional",
  "contentText": "optional",
  "images": [
    {
      "imageUrl": "xxx",
      "sortOrder": 1,
      "width": 100,
      "height": 100
    }
  ]
}
```

### 8.2 帖子列表
- **GET** `/post/list`
- 鉴权：否
- Query:
  - `sortBy`（可选）

### 8.3 帖子详情
- **GET** `/post/detail`
- 鉴权：否
- Query:
  - `id`（帖子 ID）

### 8.4 点赞
- **POST** `/post/like`
- 鉴权：是
- Body:
```json
{
  "id": 123
}
```

### 8.5 取消点赞
- **POST** `/post/unlike`
- 鉴权：是
- Body:
```json
{
  "id": 123
}
```

### 8.6 删除帖子
- **POST** `/post/delete`
- 鉴权：是
- Body:
```json
{
  "id": 123
}
```

### 8.7 新增评论
- **POST** `/post/comment/create`
- 鉴权：是
- Body:
```json
{
  "postId": 123,
  "contentText": "评论内容",
  "parentId": 0,
  "replyToUserId": 0
}
```
> `parentId`、`replyToUserId` 为可选（用于回复场景）。

### 8.8 评论列表
- **GET** `/post/comment/list`
- 鉴权：否
- Query:
  - `postId`（必需）
  - `page`（可选，默认 1）
  - `pageSize`（可选，默认 10）

### 8.9 删除评论
- **POST** `/post/comment/delete`
- 鉴权：是
- Body:
```json
{
  "id": 456
}
```

## 9. Diagnosis 模块

### 9.1 诊断列表（本人）
- **GET** `/diagnosis/list`
- 鉴权：是
- Query:
  - `page`（可选，默认 1）
  - `pageSize`（可选，默认 10，最大 50）
- 返回：
  - `list`
  - `pagination: { page, pageSize, total, totalPages }`

### 9.2 诊断详情（本人）
- **GET** `/diagnosis/detail`
- 鉴权：是
- Query:
  - `id`（诊断记录 ID）

### 9.3 发起诊断
- **POST** `/diagnosis/run`
- 鉴权：是
- Body:
```json
{
  "imageUrl": "http://... 或 uploads/...",
  "symptomText": "optional",
  "cropType": "optional"
}
```
